开服之前，请检查服务器是否安装了 Java，在终端输入

`java -version`

如果你看到类似以下的输出，那么服务器中已经安装了 Java

```txt
openjdk version "21.0.2" 2024-01-16 LTS
OpenJDK Runtime Environment Zulu21.32+17-CA (build 21.0.2+13-LTS)
OpenJDK 64-Bit Server VM Zulu21.32+17-CA (build 21.0.2+13-LTS, mixed mode, sharing)
```

## Java 安装

如果已经安装了 Java，可以忽略掉这一部分。如果环境未安装 Java 可以参考：

### 软件包管理器安装

下方以安装 `openjdk-17` 为例

<Tabs>
    <TabItem label="Debian/Ubuntu">sudo apt install openjdk-17-jdk</TabItem>
    <TabItem label="CentOS/RHEL">sudo yum install java-11-openjdk</TabItem>
    <TabItem label="Arch Linux">sudo pacman -S jdk11-openjdk</TabItem>
</Tabs>

## 多版本 Java 共存

方法很多，这里举出两种

### 方法一

<Tabs>
    <TabItem label="Debian/Ubuntu">update-alternatives --config java</TabItem>
    <TabItem label="CentOS/RHEL">update-alternatives --config java</TabItem>
    <TabItem label="Arch Linux">archlinux-java set jdk-vesion</TabItem>
</Tabs>

然后选择你想使用的 Java，接着 `java -version` 验证版本

### 方法二

设置别名

确定你所使用的终端，此处以 `bash` 和 `Azul Zulu` 为例

`vim ~/.bashrc` 在这个文件末尾追加

```bash
# java
alias java8='/usr/lib/jvm/zulu-8/bin/java'
alias java17='/usr/lib/jvm/zulu-17/bin/java'
```

然后刷新：`source ~/.bashrc`

- 使用 Java8 开服：

```bash
java8 -jar example.jar
```

- 使用 Java17 开服：

```bash
java17 -jar example.jar
```

在 `/opt` 目录中使用 `mkdir` 命令创建一个目录用于开服

![](_assets/Linux开服/开服/5.png)

将核心放入服务器内，并编写简单的开服脚本

使用 `chmod` 命令给脚本增加运行权限

```bash
chmod +x example.sh
```

![](_assets/Linux开服/开服/6.png)

输入 `./start.sh` 或者输入 `bash start.sh` 即可启动

![](_assets/Linux开服/开服/7.png)

使用 `vim` 命令修改 eula 为 `true` 就可以开服了

![](_assets/Linux开服/开服/8.png)
